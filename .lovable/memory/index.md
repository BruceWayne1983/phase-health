# Project Memory

## Core
Aesthetic: Anadya — luxury feminine, warm gold + ivory + champagne. Inspired by Augustinus Bader, La Mer, Charlotte Tilbury. NEVER period-tracker, NEVER clinical-tech, NEVER dark themes.
Gold is the hero accent (--gold, --gold-bright, --gold-deep, --gold-soft). Rose is sparing, occasional warmth only — never primary.
Typography: Cormorant Garamond (display/serif) + Inter (sans). Use `font-display` for big headings.
Logo: src/assets/anadya-logo.jpeg is the master brand asset and source of truth for colour temperature. Use `<Wordmark variant="full" />` for hero placements.
Decorative system: minimal SVG ornaments only — OrnamentDivider, GoldRule, HormoneCurve. No florals/laurels/sun motifs.
Desktop layouts must be FULL-BLEED IMMERSIVE — gold halos, framed logo plates, glass cards floating over the canvas.
Primary CTA = `.btn-primary` (gold gradient on cream text). Secondary = `.btn-ghost` (gold border).
Copy rules: British English, no em dashes, sentence case for buttons/headers. Forbidden words in user-facing copy: ritual, sanctuary, goddess, sacred, rise, awaken, journey. No emoji in UI. Buttons: Continue / Begin / Save / Back.
App name: ANADYA. Tagline: "Hormone health, every phase." Hero: "Your body, understood."
